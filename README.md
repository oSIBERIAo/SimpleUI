# 简UI————VUE UI组件库

[官方文档](https://github.com/oSIBERIAo/SimpleUI)

简洁明了的 UI 框架。

## 项目特点

1. 使用 Travis CI 进行持续集成
2. 有丰富的单元测试，项目完成时，期望测试覆盖率超过 90%
3. 自说明的代码，即使没有注释，你也能看懂
4. 我使用 parcel 构建方便上手，后期改为 Vue Cli 3 以实现更多功能

## 背景

学习VUE过程中造的 UI 轮子。

## 视觉稿

本 UI 框架借鉴了一些成熟 UI 框架的外观。

## 开始

1. 安装依赖
    ```
    yarn install
    ```

2. 启动 dev server
    ```
    yarn serve
    ```
